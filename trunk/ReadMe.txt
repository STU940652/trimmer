TRIMMER README
==============

Trimmer is released under LGPL 2.1.  It is currently hosted at
http://code.google.com/p/trimmer/, where you can find the most 
complete documentation.


DEPENDENCIES
============
  Running the binary
    - VLC

  Running the Python
    - Python 3.x
    - wxPython

  Building the binary
    - cx_freeze 2.x
